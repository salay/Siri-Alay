Sensibly


Sensibly is an Acuity-like scheduling app for therapists.... (that might've devolved into an app to schedule time with therapy dogs).

Link to Sensibly Website deployed on Heroku
Link to Github for Sensibly

Technologies Used:

  • Python
  • WTForms
  • Flask
  • Jinja
  • SQLite
  • PostgreSQL
  • Vanilla JS
  • Bootstrap

Features:

  • Flask Authentication with Validation (Register for Account and Login)
  • Public Therapist Profile Page & Private User Profile Page.
  • Vanilla JS calendar
  • Cannot schedule appointments if not signed in.
  • User can click on a date, make an appointment, view it on the same page, edit it, and delete it.
  • If signed in as a user, you cannot view therepists schedules.
  • Therapists can view only their own schedules when signed in as a therapist.
  • Users cannot schedule an appointment in the past or during a slot in which an appointment is already scheduled with said therapist.
  • Flexible Search Bar on therapists list page.
